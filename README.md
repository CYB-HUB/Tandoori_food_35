5 Sq. Sarah Bernhardt, 35200 Rennes — 02 76 10 37 57

## Checklist — Mise en ligne & Proposition au restaurant

### 🌐 1. Hébergement & Domaine

| Élément | Option gratuite | Option payante |
|---|---|---|
| **Nom de domaine** | — | `tandoorifood35.fr` (~10€/an sur OVH/Gandi) |
| **Hébergement** | GitHub Pages (gratuit) | Vercel / Netlify (gratuit aussi) |
| **SSL (HTTPS)** | Inclus GitHub Pages | Inclus partout |

> **Recommandation :** GitHub Pages suffit pour démarrer, acheter le nom de domaine en priorité.

---

### 📸 2. Contenu à récupérer auprès du restaurant

- [ ] Photos des plats en haute qualité (minimum 10)
- [ ] Photos de la salle / devanture
- [ ] Menu complet et à jour avec tous les prix exacts
- [ ] Logo du restaurant (si existant)
- [ ] Horaires exacts par jour (notamment jours fériés)
- [ ] Lien Uber Eats exact du restaurant
- [ ] Réseaux sociaux (Instagram, Facebook s'ils en ont)

---

### ⚖️ 3. Légal (obligatoire en France)

- [ ] **Mentions légales** — nom du propriétaire, SIRET, hébergeur
- [ ] **Politique RGPD** — si Google Analytics ou formulaire de contact
- [ ] **Bandeau cookies** — si tracking / analytics

---

### 🔧 4. Améliorations techniques avant lancement

- [ ] **Google Maps API key** — remplacer l'iframe générique par une vraie clé
- [ ] **Vraies photos** — intégrer les photos du restaurant dans la galerie
- [ ] **Formulaire de contact** fonctionnel (Formspree ou EmailJS — gratuit)
- [ ] **SEO** — meta description, Open Graph (partage réseaux sociaux), favicon
- [ ] **Google Analytics** — tracker les visites pour montrer les stats au restaurant
- [ ] **Schema.org Restaurant** — balisage JSON-LD pour référencement local Google

---

### 📊 5. Référencement local

- [ ] Lier le site à la fiche **Google Business Profile** du restaurant
- [ ] Ajouter l'URL sur la page **Uber Eats** du restaurant
- [ ] Soumettre à **Google Search Console**

---

### 💼 6. Proposition commerciale au restaurant

- [ ] **Démo en ligne** — montrer le site sur leur téléphone / tablette
- [ ] **PDF de présentation** — captures d'écran + liste des fonctionnalités
- [ ] **Proposition tarifaire** :
  - Création du site : 150–400 € (one-shot)
  - Maintenance mensuelle : 20–50 €/mois (mises à jour menu, hébergement)
  - Nom de domaine : ~10 €/an (refacturé au client)
- [ ] **Contrat simple** — prestation, délais, droits sur les visuels

---

### ✅ Ordre de priorité

```
1. Activer GitHub Pages → site en ligne immédiatement (gratuit)
2. Acheter le domaine tandoorifood35.fr
3. Récupérer photos + menu exact auprès du restaurant

```

## Stack technique

- HTML / CSS / JavaScript — zéro dépendance externe
- Google Fonts (Inter)
- Hébergement : GitHub Pages

## Fonctionnalités du site

- Badge **Ouvert / Fermé** calculé en temps réel
- **Menu interactif** avec filtres par catégorie animés
- **Carrousel avis** automatique (défilement toutes les 4s)
- **Tableau horaires** avec highlight du jour actuel
- Carte Google Maps embarquée
- Responsive mobile complet
- Navbar glassmorphism avec opacité au scroll
